My project

My personal Training Website

This HTML project showcases my early web development skills through a personal training website I created. The website features a clean, powder blue background with styled headings and content sections that introduce myself as a computer science student at UTM. It includes formatted text with different colors and sizes, lists of my technical skills and favorite drinks, as well as examples of chemical formulas using subscript and superscript tags. The site also contains inspirational quotes in a blockquote format, external links to social media platforms, and full contact information with an interactive email button. While demonstrating fundamental HTML elements like headings, paragraphs, lists, and images, the project also incorporates basic styling through inline CSS, showing my initial understanding of web design principles. The website serves as both a personal profile and a training-themed platform, highlighting my ability to structure content and apply simple formatting in web development.

Grid Calculator

This project demonstrates my ability to create an interactive calculator using HTML and CSS Grid layout. I built a fully structured calculator interface with a clean, organized design that includes all essential calculator functions. The HTML document features a responsive input display at the top, followed by a grid of operation buttons (addition, subtraction, multiplication, and division) and numbered buttons (0-9) arranged in a logical calculator layout. I also implemented additional functionality buttons including a decimal point, an all-clear (AC) button, and an equals button for calculations. The semantic HTML structure with proper class and ID naming conventions shows my attention to detail in web development. While this file represents the HTML foundation, it's designed to work with an external CSS stylesheet to create a visually appealing and functional grid-based calculator that could be enhanced with JavaScript for full functionality. The commented reference to CSS-Tricks demonstrates my commitment to learning best practices in CSS Grid implementation.
